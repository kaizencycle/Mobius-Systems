# Kaizen-OS Branding Assets

This directory contains the visual identity assets for Kaizen-OS documentation and branding.

## Logo Files

### `logo/kaizen-logo.svg`
Main Kaizen-OS logo featuring:
- **Deep Teal** (#2C4F5E) background circle representing contemplation and depth
- **Warm Beige** (#D4C5A9) vessel shape representing the system
- **Gold** (#FFD700) Kintsugi repair seams representing healing and integrity
- Central node and outer nodes representing connected systems

**Usage**: Header logo, documentation branding, official communications

**Dimensions**: 120x120px (scalable SVG)

### `logo/kaizen-favicon.svg`
Simplified favicon version of the logo for browser tabs.

**Usage**: Browser favicon, app icons

**Dimensions**: 32x32px (scalable SVG)

## Color System

Based on the Kaizen Visual Design System v1.0.0:

### Primary Colors
- **Deep Teal** `#2C4F5E` - Primary background, contemplation
- **Darker Teal** `#1A2F3A` - Depth, shadows
- **Lighter Teal** `#3D5A6B` - Highlights, clarity

### Accent Colors
- **Gold** `#FFD700` - Healing, accent, emphasis
- **Gold Dark** `#B8860B` - Gold variant for hover states

### Text Colors
- **Cream** `#F5E6D3` - Primary text, illumination
- **Warm Beige** `#D4C5A9` - Secondary text, ceramic

### Semantic Colors
- **Success** `#95e1d3` - Mint (healing, growth)
- **Warning** `#ffd93d` - Amber (attention, caution)
- **Error** `#ff6b6b` - Coral Red (fracture, critical)
- **Neutral** `#666666` - Gray (deprecated, inactive)

## Typography

### Headings
- **Serif fonts** (Georgia, Merriweather, Crimson Text) for philosophical gravitas
- **Sans-serif fonts** (Inter, Source Sans Pro) for clarity and readability

### Body Text
- **Sans-serif** (Inter recommended) for maximum readability

### Code
- **Monospace** (JetBrains Mono, Fira Code) for technical content

## Design Principles

1. **Kintsugi Philosophy**: Visual representation of healing and repair
2. **Circular Composition**: Completeness, cycles, wholeness
3. **Generous Negative Space**: Contemplation and focus
4. **Golden Accents**: Healing, value, illumination

## Usage Guidelines

### Logo Usage
- Maintain aspect ratio when scaling
- Minimum size: 32px height
- Provide adequate clear space (minimum 20% of logo height)
- Use gold (#FFD700) for light backgrounds
- Use cream (#F5E6D3) for dark backgrounds

### Color Usage
- Use Deep Teal as primary background
- Use Gold sparingly for emphasis and CTAs
- Maintain sufficient contrast ratios (minimum 4.5:1 for text)
- Test with colorblindness simulators

### Accessibility
- All images require alt text
- Maintain WCAG AA contrast standards
- Test with screen readers
- Provide text alternatives for complex graphics
