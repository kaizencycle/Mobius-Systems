## RFC Status Definitions

- **Draft**: Work in progress, not ready for review
- **Proposed**: Ready for community discussion
- **Active**: Under formal review by Council/TSC
- **Accepted**: Approved, awaiting implementation
- **Implemented**: Shipped in a release
- **Rejected**: Not accepted (with rationale)
- **Superseded**: Replaced by a newer RFC
- **Deferred**: Tabled for future consideration

## How to Propose an RFC

See [RFC Process](../FOUNDATION/PROCESS/RFC_PROCESS.md) for detailed instructions.

**Quick Start**:
1. Copy `rfcs/RFC-0000-template.md`
2. Write your proposal
3. Submit PR to reserve RFC number
4. Announce in GitHub Discussions
5. Iterate based on feedback
6. Request formal review from TSC/Council
